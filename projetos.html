Projetos

Aqui estarei disponibilizando alguns projetos que se encontram hospedados na web.

  • Projeto Barbearia Eugod

    Projeto do site de uma barbearia, que contem três páginas. Desenvolvido utilizando apenas HTML e CSS.

  • Projeto Robotron 2000

    Projeto para a "construção" de um robô que irá combater alienigenas, neste projeto a pessoa pode escolher quantas peças irá colocar no robô, assim alterando os seus status dinâmicamente. Desenvolvido utilizando HTML, CSS e JavaScript.

  • Projeto Número Secreto

    Projeto de um jogo que consiste em advinhar um número que foi sorteado, você deverá falar em seu microfone um número entre 0 e 100 até acertar o número sorteado. Desenvolvido utilizando HTML, CSS, JavaScript e API Web Speech.

Você poderá acessar o código de cada um desses projetos, e muitos outros que ainda não estão hospedados na web, visitando o meu GitHub
