Projetos

Aqui estarei disponibilizando alguns projetos que se encontram hospedados na web.

  • Projeto Barbearia Eugod

    Projeto do site de uma barbearia, que contem três páginas. Desenvolvido utilizando apenas HTML e CSS.

  • Projeto Robotron 2000

    Projeto para a "construção" de um robô que irá combater alienigenas, neste projeto a pessoa pode escolher quantas peças irá colocar no robô, assim alterando os seus status dinâmicamente. Desenvolvido utilizando HTML, CSS e JavaScript.

  • Projeto Número Secreto

    Projeto de um jogo que consiste em advinhar um número que foi sorteado, você deverá falar em seu microfone um número entre 0 e 100 até acertar o número sorteado. Desenvolvido utilizando HTML, CSS, JavaScript e API Web Speech.

  • Projeto Mochila de Viagem

    Projeto em que você pode adicionar e remover itens em uma lista. Desenvolvido utilizando HTML, CSS e JavaScript.

  • Projeto Meteora

    Projeto feito para treinar os conhecimentos em Bootstrap5 adquiridos no curso da Alura. Deselvolvido utilizando HTML, CSS e Bootstrap.

  • Projeto Serenatto

    Projeto feito para treinar os conhecimentos em Bootstrap5 adquiridos no curso da Alura. Deselvolvido utilizando HTML, CSS, Bootstrap e JavaScript.

  • Projeto EuBank

    Projeto feito com o intuito de treinar os conhecimentos adquiridos durente um curso feito na Alura. Consiste na validação dos campos e dados de um formulário de cadastro.

  • Projeto Rick and Morty

    Projeto feito durante as aulas da matéria de Desenvolvimento de Sistemas no SENAI, usando React com Vite. Esse projeto tem como objetivo consumir os dados de uma API e apresenta-los na tela.

Você poderá acessar o código de cada um desses projetos, e muitos outros que ainda não estão hospedados na web, visitando o meu GitHub
